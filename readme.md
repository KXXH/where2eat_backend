## 使用方法

`pip install -r requirements.txt `

### 开启api服务

`python api.py`

### 初始化数据库

虽然开启api服务时已经初始化数据库，但是没有账户和密码，没法登陆。

修改`where2eatdb.db`，插入你自己的用户名和密码（目前均为明文）。
